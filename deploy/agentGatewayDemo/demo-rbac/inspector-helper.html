QwenPaw RBAC 演示 — 双屏 Inspector 助手

Gateway: http://localhost:3000/mcp · Transport: Streamable HTTP · Custom Header: Authorization（可选）

无 Token

Inspector 中不要添加 Authorization Header，直接 Connect。 List Tools 预期: 仅 forum_list_posts（读论坛）。

普通员工 Token（employeeQwenpaw）

List Tools 预期: forum_list_posts + forum_create_post

管理员 Token（managerQwenpaw）

List Tools 预期: 5 个工具（人事 + 论坛全部）

权限矩阵（三档）

工具（Inspector 名）无 Token普通员工 Token管理员 Token
forum_list_posts允许允许允许
forum_create_post拒绝允许允许
forum_delete_post拒绝拒绝允许
hr_get_employee拒绝拒绝允许
hr_update_employee拒绝拒绝允许

Inspector 演示流程

  1. 无 Token：不填 Authorization → Connect → List Tools 仅 1 个 → Call forum_list_posts 成功
  2. 普通员工 Token：粘贴后 Reconnect → List Tools 2 个 → Call forum_create_post 成功
  3. 管理员 Token：粘贴后 Reconnect → List Tools 5 个 → Call hr_get_employee 成功
  4. 攻击对比（普通员工 Token）：Call hr_get_employee {} → Unknown tool（RBAC 拦截）
